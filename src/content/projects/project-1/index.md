Rate A Repository is a React Native (Expo) app that lets users search
GitHub repositories, view their details, and rate them. Built to practice
API integration, navigation, and form handling in a mobile context.

## 📋 Features

- ✅ Search and browse GitHub repositories
- ✅ View repository details (stars, forks, description)
- ✅ Rate and review repositories
- ✅ Built with Expo

## 🛠️ Stack

React Native, Expo, TypeScript
